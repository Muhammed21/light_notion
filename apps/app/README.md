# App Expo

Application mobile/web React Native avec Expo.

## Lancer l'application

1. Installer les d√©pendances (depuis la racine du monorepo)
   ```bash
   pnpm install
   pnpm build
   ```

2. Lancer l'app
   ```bash
   pnpm ios      # pour iOS Simulator
   pnpm android  # pour Android Emulator
   ```

3. Ouvrir l'app sur :
   - iOS Simulator : appuyer sur `i`
   - Android Emulator : appuyer sur `a`
   - Web : appuyer sur `w`
   - Expo Go : scanner le QR code
