# Payload CMS

Backend CMS avec Payload et Next.js.

## Lancer Payload

1. Installer les dépendances (depuis la racine du monorepo)
   ```bash
   pnpm install
   pnpm build
   ```

2. Copier les variables d'environnement
   ```bash
   cp .env.example .env
   ```

3. Lancer le serveur
   ```bash
   pnpm dev
   ```

4. Ouvrir http://localhost:3000/api et créer le premier utilisateur admin
